# Mapa de rutas inicial

- `/login`: acceso con Supabase o modo demo por rol.
- `/dashboard`: centro de operaciones y alertas.
- `/clientes`: maestro de empresas y centros.
- `/activos`: parque instalado.
- `/ordenes`: alta rápida y listado de OT.
- `/planificacion`: planner operativo por técnico.
- `/legalizaciones`: vencimientos y expedientes.
- `/kpi`: panel ejecutivo inicial.
- `/tecnico`: consola móvil/PWA.
- `/api/health`: comprobación simple de servicio.
- `/api/reports/work-order/[id]`: PDF demo del parte de trabajo.
